Key Features:

User Authentication: Secure user registration and login functionality using JWT for token-based authentication.
Task Management: Users can create, update, and delete tasks. Each task includes details like name, duration, due date, and priority.
Task Filtering: Easily switch between viewing pending and completed tasks with a simple toggle.
Responsive Design: Optimized for both desktop and mobile devices, ensuring a consistent experience across different screen sizes.
Modals for Task Details: Clean and user-friendly modals for adding and editing tasks, ensuring a smooth user interaction.
Tech Stack:

Frontend: React for building interactive UI components and managing application state. CSS for styling and ensuring responsive design.
Backend: Node.js and Express for creating RESTful APIs. JWT for handling user authentication securely.
Database: MySQL for managing and storing user and task data.
Axios: For making HTTP requests from the frontend to the backend API.
Challenges and Learning Outcomes:

Working on Taskify has been an incredible learning experience. I faced and overcame several challenges, such as implementing secure authentication, ensuring seamless data flow between frontend and backend, and optimizing the application for performance. This project enhanced my understanding of full-stack development and improved my skills in building scalable web applications.

Future Enhancements:

Offline Mode: Implementing offline functionality so users can manage tasks without an internet connection.
Notifications: Adding reminders and notifications for upcoming due tasks.
Collaboration: Enabling task sharing and collaboration among multiple users.
